This is the FVA-BW (http://www.fva-bw.de/) version of the open source tree growth simulator BWINPro from NW-FVA (https://www.nw-fva.de/).

It is currently based on Version 7.8 of BWINPro. The current upstream version can be found at https://www.nw-fva.de/?id=194

The most notable features of this fork to date are:

* JOGL/JOGAMP-based integrated Java 3D support
* Updated and improved english and german translations
* Progress indication of MS Access batch mode
* gradle build system

Future plans include:

* climate-sensitive growth simulation
* more flexible batch mode settings
